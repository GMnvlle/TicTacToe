PROJECT TITLE: TicTacToe
PURPOSE OF PROJECT: This tic-tac-toe game is programmed in java and is a game of tic-tac-toe. The player takes turns entering a number value corresponding with an open spot on the board. This project allowed me to solve more complex programming problems like having to work with deeply embedded blocks.
VERSION or DATE: December 2025
HOW TO START THIS PROJECT: Create a Game object from the Game class and run the play() method.
